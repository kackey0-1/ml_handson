正規線形モデル紹介¶

分散分析と回帰分析を一つのモデルにまとめたものを正規線形モデルという

正規線形モデルを利用することで、より多くの説明変数を扱うことが可能になる

  • 例:
    • カテゴリ変数: 薬の有無
    • 連続変数: 魚の体長

import statsmodels.api as sm
from statsmodels.formula.api import ols
from sklearn.linear_model import LinearRegression
model_ols = ols('length ~ food + medicine', data=df).fit()
model_ln = LinearRegression().fit(df[['food', 'medicine']], df['length'])
# anova_table = sm.stats.anova_lm(model, typ=2)
anova_table = sm.stats.anova_lm(model_ols, typ=1)
print(anova_table)
